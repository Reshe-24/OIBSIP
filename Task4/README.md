# Project Title
Online Examination System
# Description
The Online Examination System is a Java Swing-based desktop application developed to conduct online quizzes and examinations. The application provides a graphical user interface (GUI) for user login, answering multiple-choice questions, score calculation, password updating, and timer-based exam submission.
This project is useful for understanding:
- Java Swing GUI
- Event Handling
- Object-Oriented Programming
- Timer Handling in Java
- Multiple Choice Question System
- Basic Online Examination Logic
# Features
## Login Authentication
- Secure login screen
- Default credentials:
Username: admin  
Password: 1234
---
## Online Exam Module
Users can:
- Start the examination after login
- Answer multiple-choice questions
- Navigate through questions using the Next button
- Submit the exam anytime
---
## Timer Functionality
- Countdown timer for the exam
- Automatically submits exam when time is over
---
## Score Calculation
- Calculates score based on correct answers
- Displays final score after submission
---
## Password Update
Users can:
- Change login password during the exam
---
## Logout Functionality
Users can:
- Logout from the examination system anytime
# Technologies Used
- Java
- Java Swing
- AWT Event Handling
- Timer Class
