# Project Title
Number Guessing Game
# Description
The Number Guessing Game is a Java Swing-based desktop application developed to provide an interactive number guessing experience. The application generates a random number between 1 and 100, and the player must guess the correct number within limited attempts.
This project is useful for understanding:
- Java Swing GUI
- Event Handling
- Random Number Generation
- Object-Oriented Programming
- Basic Game Logic
# Features
## Random Number Generation
- Generates a random number between 1 and 100
- New random number generated for every round
---
## Guess Checking System
Users can:
- Enter a number guess
- Check whether the guess is correct
- Receive hints like:
  - Too High
  - Too Low
---
## Attempts Management
- Player gets 10 attempts per round
- Attempts decrease after every guess
- Game ends when attempts reach zero
---
## Score System
- Points are awarded based on remaining attempts
- Higher score for fewer guesses
- Score updates automatically
---
## Multiple Rounds
Users can:
- Start a new round
- Continue playing without losing total score
- Track current round number
# Technologies Used
- Java
- Java Swing
- AWT Event Handling
- Random Class
